      ArtWork Class

      The ArtWork class is responsible for creating and drawing various shapes on the output screen, such as lines, squares, circles, rectangles, and triangles. It also contains various properties and methods for managing the state of the drawn shapes, such as fill, pen, and position.

      Remarks

      The class contains a ReaderWriterLockSlim object called '_rwLock' which is used to provide thread-safety while accessing the list of shapes.
